## Clone the repository
```bash
git clone https://github.com/raspberri05/persona.fm.git
``` 

This creats a local copy of the repository on your machine

## Install dependencies
<Tabs>
  <TabItem label="npm">
    ```bash
    cd application/ && npm install
    ```
  </TabItem>
  <TabItem label="pnpm">
    ```bash
    cd application/ && pnpm install
    ```
  </TabItem>
  <TabItem label="bun">
    ```bash
    cd application/ && bun install
    ```
  </TabItem>
</Tabs>
