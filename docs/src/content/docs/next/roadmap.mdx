We are currently working towards implementing basic features to allow for a public release of version 1.0.0. Below you can find the features that you can expect to see in our first release:

## Features to be included in v1.0.0
* Ability to your last.fm account to your persona.fm account to be able to generate music "personas" based off your listening history for the past week
* Ability to view past generated personas
* Google authentication
* Ability to edit user profile
* Ability to delete persona.fm account
* Implementation of rate limits to prevent abuse of the application
